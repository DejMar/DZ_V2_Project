1. Pregled aplikacije

Dom Zdravlja je web aplikacija za evidenciju lijekova u domu zdravlja. Aplikacija radi potpuno lokalno — podaci se čuvaju u JSON fajlovima, bez baze podataka, Dockera ili internetske veze.

Izgrađena je u .NET 8 sa Blazor Server tehnologijom. Korisnički interfejs se prikazuje u web browseru, a logika se izvršava na lokalnom računaru.

Models/ Entiteti: Lijek, Korisnik, Zahtjev, Ambulanta
Services/ Poslovna logika i čitanje/pisanje JSON fajlova
Components/ Blazor UI stranice po ulogama
Data/ users.json, medicines.json, ambulances.json, requests.json

2. Uloge korisnika

Uloga Ko je? Šta može?
Administrator Upravlja zalihama u domu zdravlja Unosi/uređuje/briše lijekove, upravlja korisnicima, pregleda i izvozi izvještaje
Moderator Zadužen za distribuciju lijekova Pregleda zahtjeve, odobrava, odbija i izdaje lijekove (ne može izdati istekle)
Korisnik Doktor u ambulanti Šalje zahtjeve za lijekove za svoju ambulantu, prati status zahtjeva

3. Tok rada (workflow)

Osnovni proces od zahtjeva do izdavanja lijeka:

Korisnik šalje zahtjev
→
Moderator odobrava ili odbija
→
Moderator izdaje lijek
→
Zaliha se smanjuje
→
Admin prati izvještaje

4. Pokretanje aplikacije

Preduvjet

Instaliran .NET 8 SDK. Provjera:

dotnet --version

Koraci

cd DomZdravlja
dotnet run

Otvori u browseru (Chrome preporučen):

http://localhost:5141/prijava

Napomena: Koristi http:// a ne https:// u development modu. Port može biti drugačiji — provjeri terminal nakon dotnet run.

6. Nove funkcionalnosti

6.1 Upravljanje korisnicima

Administrator može na stranici /admin/korisnici:

  • dodati novog korisnika (ime, username, lozinka, uloga, ambulanta)
  • urediti postojećeg korisnika
  • deaktivirati ili ponovo aktivirati nalog

Deaktivirani korisnici se ne mogu prijaviti u aplikaciju.

6.2 Rok trajanja lijekova

Svaki lijek ima polje Rok trajanja. Sistem automatski prikazuje upozorenja:

  • Ističe uskoro — rok ističe u narednih 30 dana
  • Istekao — rok je prošao; moderator ne može izdati takav lijek

Demo podaci: Hlorheksidin je istekao, Amoxicillin ističe uskoro.

6.3 Izvoz izvještaja

Na stranici Izvještaji administrator može:

  • Preuzmi CSV — izvještaj za Excel (zalije, zahtjevi, statistika)
  • Izvještaj (PDF) — otvara štampani HTML izvještaj (Ctrl+P → Sačuvaj kao PDF)

Export zahtijeva prijavu kao administrator (session cookie).

8. Test scenariji

Sljedeći scenariji pokrivaju cjelokupan tok aplikacije. Preporučeno je testirati ih redom.

Test 1 — Prijava i kontrola pristupa

  1. Otvori http://localhost:5141/prijava
  2. Unesi pogrešne podatke (npr. admin / pogresna) i klikni Prijavi se
  3. Unesi ispravne podatke: admin / admin123
  4. Pokušaj ručno otvoriti /moderator/zahtjevi dok si prijavljen kao admin
Očekivano: Pogrešna prijava prikazuje grešku. Ispravna prijava vodi na stranicu Lijekovi. Admin nema pristup moderator stranici (ili vidi samo svoje opcije u meniju).

Test 2 — Administrator: upravljanje lijekovima

  1. Prijavi se kao admin / admin123
  2. Na stranici Lijekovi klikni + Dodaj lijek
  3. Unesi: Naziv = Vitamin C 500mg, Jedinica = tableta, Količina = 200, Min. zaliha = 50, Rok trajanja = za godinu dana
  4. Klikni Spremi
  5. Uredi postojeći lijek — promijeni količinu
  6. Idi na Izvještaji i provjeri da li se novi lijek pojavljuje
Očekivano: Novi lijek se pojavljuje u tabeli. Izmjene se čuvaju. Izvještaj prikazuje ažurirano stanje zaliha.

Test 3 — Korisnik: slanje zahtjeva

  1. Odjavi se (klikni Odjava)
  2. Prijavi se kao korisnik1 / user123
  3. Idi na Novi zahtjev
  4. Odaberi lijek Paracetamol 500mg, količina = 20
  5. Klikni Pošalji zahtjev
  6. Idi na Moji zahtjevi
Očekivano: Prikazana ambulanta „Opća ambulanta". Zahtjev se pojavljuje sa statusom Na čekanju.

Test 4 — Moderator: odobravanje i izdavanje

  1. Odjavi se i prijavi kao moderator / mod123
  2. Na stranici Zahtjevi pronađi zahtjev od korisnik1
  3. Klikni Odobri
  4. U sekciji „Odobreni" klikni Izdaj lijek
  5. Provjeri historiju — status treba biti Izdato
Očekivano: Zahtjev prelazi: Na čekanju → Odobren → Izdato. Zaliha Paracetamola se smanjuje za 20 (provjeri kao admin u Izvještajima).

Test 5 — Moderator: odbijanje zahtjeva

  1. Prijavi se kao korisnik2 / user123
  2. Pošalji zahtjev za Ibuprofen 400mg, količina = 10
  3. Odjavi se, prijavi kao moderator
  4. Klikni Odbij na tom zahtjevu
  5. Prijavi se ponovo kao korisnik2 i provjeri Moji zahtjevi
Očekivano: Status zahtjeva je Odbijen. Zaliha Ibuprofena se ne mijenja.

Test 6 — Rok trajanja lijekova

  1. Prijavi se kao admin
  2. Na stranici Lijekovi provjeri status Hlorheksidin — treba biti Istekao
  3. Provjeri Amoxicillin — treba biti Ističe uskoro
  4. Idi na Izvještaje — provjeri brojače „Istekli" i „Ističe uskoro"
  5. Prijavi se kao moderator, pokušaj izdati istekli lijek (ako postoji odobren zahtjev za Hlorheksidin)
Očekivano: Istekli lijek ima crvenu oznaku. Dugme „Izdaj lijek" je onemogućeno za istekle lijekove.

Test 7 — Upravljanje korisnicima

  1. Prijavi se kao admin
  2. Idi na Korisnici
  3. Klikni + Dodaj korisnika — unesi: Ime = Dr. Test, Username = testuser, Lozinka = test123, Uloga = Korisnik, Ambulanta = Opća ambulanta
  4. Odjavi se i prijavi kao testuser / test123
  5. Prijavi se ponovo kao admin, deaktiviraj testuser
  6. Pokušaj se prijaviti kao testuser
Očekivano: Novi korisnik se može prijaviti. Nakon deaktivacije prijava ne uspijeva.

Test 8 — Izvoz izvještaja

  1. Prijavi se kao admin
  2. Idi na Izvještaji
  3. Klikni Preuzmi CSV — otvori fajl u Excelu ili editoru
  4. Klikni Izvještaj (PDF) — u browseru Ctrl+P → Sačuvaj kao PDF
Očekivano: CSV sadrži zalihe, statistiku i zahtjeve po ambulantama. HTML izvještaj je formatiran za štampu.

Test 9 — Cjelokupan end-to-end tok

  1. Admin provjeri početne zalihe u Izvještajima (zapamti količinu Paracetamola)
  2. Korisnik3 (korisnik3 / user123) pošalje zahtjev za 30 tableta Paracetamola
  3. Moderator odobri i izda zahtjev
  4. Admin provjeri da je zaliha Paracetamola smanjena za 30
  5. Admin provjeri Izvještaj „Zahtjevi po ambulantama" — Stomatološka ambulanta ima +1 izdato
Očekivano: Cijeli lanac radi: zahtjev → odobrenje → izdavanje → ažurirana zaliha → ažuriran izvještaj.

9. Struktura podataka (JSON)

Podaci se automatski kreiraju u folderu DomZdravlja/Data/ pri prvom pokretanju:

FajlSadržaj
users.jsonKorisnici, lozinke, uloge, ambulanta, isActive (aktivan/deaktiviran)
medicines.jsonLijekovi, količine, minimalne zalihe, expiryDate (rok trajanja)
ambulances.jsonLista ambulanti u domu zdravlja
requests.jsonSvi zahtjevi za lijekove i njihovi statusi

Početni lijekovi (sa rokom trajanja)

  • Paracetamol 500mg — 500 tableta (rok: +18 mjeseci)
  • Ibuprofen 400mg — 300 tableta (rok: +12 mjeseci)
  • Amoxicillin 500mg — 150 kapsula (rok: +20 dana — ističe uskoro)
  • Hlorheksidin — 40 flaša (rok: prošao — istekao)
  • Fizološki rastvor — 200 vrećica (rok: +24 mjeseca)
